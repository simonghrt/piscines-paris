# Piscine Paris app

This app has been created with Ionic and uses the services of the Paris city.

### Requirements

Npm
Ionic

### Installation

``` bash
npm install
```

### Run

##### Web

``` bash
# To run the code in your browser
ionic serve
```

##### Android phone

Follow the instructions on the Android website
